# Converting Transcript Template to Google Docs

**Purpose**: Instructions for using the homeschool transcript template in Google Docs

**Time Required**: 10-15 minutes

**Last Updated**: February 14, 2026

---

## Option 1: Convert Existing DOCX to Google Docs

If you already have the transcript template as a Microsoft Word (.docx) file:

### Step 1: Upload to Google Drive

1. Go to [Google Drive](https://drive.google.com)
2. Click **+ New** button (top left)
3. Select **File upload**
4. Choose your transcript template DOCX file
5. Wait for upload to complete

### Step 2: Open with Google Docs

1. Find the uploaded file in Google Drive
2. Right-click on the file
3. Select **Open with → Google Docs**
4. Google will automatically convert the file

### Step 3: Make a Copy for Editing

1. With the converted document open, click **File → Make a copy**
2. Name it: "[Student Name] - Official Transcript - DRAFT"
3. Click **Make a copy**
4. You now have an editable Google Docs version

## Option 2: Create from Markdown Template

If you're starting from the markdown template file:

### Step 1: Convert Markdown to Google Docs

1. Open the `transcript-template.md` file in a text editor
2. Copy all content
3. Go to [Google Docs](https://docs.google.com)
4. Click **+ Blank** to create a new document
5. Paste the content
6. Format manually (see formatting instructions below)

### Step 2: Format the Document

Apply professional formatting:

**Fonts**:
- **Headings**: Arial or Times New Roman, 14-16pt, Bold
- **Body text**: Arial or Times New Roman, 11-12pt
- **Tables**: Arial or Times New Roman, 10-11pt

**Spacing**:
- **Line spacing**: 1.15 or 1.5
- **Paragraph spacing**: 6pt after paragraphs
- **Margins**: 1 inch on all sides (default)

**Tables**:
1. Select course listings
2. Click **Insert → Table**
3. Create tables with proper columns
4. Add borders: **Table → Table properties → Table border** (1pt black)

**Section Headings**:
1. Select heading text
2. Click **Format → Paragraph styles → Heading 1 or Heading 2**
3. Apply bold formatting

## Formatting Checklist for Google Docs

Use this checklist to ensure professional formatting:

- [ ] **Font**: Times New Roman or Arial, 11-12pt
- [ ] **Margins**: 1 inch on all sides
- [ ] **Headings**: Bold and larger font (14-16pt)
- [ ] **Section breaks**: Clear visual separation between sections
- [ ] **Tables**: Properly formatted with borders for course listings
- [ ] **Alignment**: Left-aligned for most content, centered for transcript title
- [ ] **Page numbers**: Insert page numbers (optional but recommended)
- [ ] **Line spacing**: Consistent throughout (1.15 or 1.5)
- [ ] **No extra spaces**: Remove excessive blank lines

## Filling Out the Template in Google Docs

### Step 1: Replace Placeholders

1. Use **Edit → Find and replace** (Ctrl+H / Cmd+Shift+H)
2. Find: `[Student Name]`
3. Replace with: `[Student Name]`
4. Click **Replace all**
5. Repeat for other placeholders:
   - `[MM/DD/YYYY]` → Actual dates
   - `[Homeschool Name]` → Your homeschool name
   - `[Course Title]` → Actual course names
   - `[Grade]` → Actual grades
   - `[Credits]` → Actual credits

### Step 2: Complete Course Tables

1. Click into each table
2. Fill in course titles, levels, credits, grades
3. Use **Table → Insert row above/below** to add more courses
4. Delete unused rows if needed

### Step 3: Calculate GPA

1. Use the grading scale provided
2. Calculate quality points for each course
3. Sum quality points and divide by total credits
4. Fill in GPA fields (weighted and unweighted)

### Step 4: Add Test Scores and Activities

1. Complete the standardized test scores section
2. List AP courses taken
3. Add dual enrollment courses
4. Document honors, awards, extracurriculars

### Step 5: Sign and Date

1. Print the document
2. Sign in the verification section
3. Scan back into PDF (or use digital signature tool)

## Sharing and Permissions

### Share with Universities

**Option A: Download as PDF** (Recommended):
1. Click **File → Download → PDF Document (.pdf)**
2. Save with filename: `Student_Name_Official_Transcript.pdf`
3. Upload to Common App or Coalition App
4. Email to universities if requested

**Option B: Share Google Docs Link**:
1. Click **Share** button (top right)
2. Change access to **Anyone with the link can view**
3. Copy link
4. Share link with universities (less common)

### Backup and Version Control

1. **Make copies** before major edits:
   - `[Student Name] Transcript - v1.0`
   - `[Student Name] Transcript - v2.0 (Updated Fall 2026)`

2. **Download regular backups**:
   - **File → Download → Microsoft Word (.docx)**
   - Save to your computer or external drive

3. **Enable version history**:
   - Google Docs automatically saves versions
   - Access via **File → Version history → See version history**

## Troubleshooting

### Problem: Tables don't format correctly

**Solution**:
- Manually recreate tables in Google Docs
- Click **Insert → Table**
- Set proper number of rows and columns
- Copy and paste content cell by cell

### Problem: Formatting looks different after conversion

**Solution**:
- Manually adjust fonts, spacing, and margins
- Use **Format → Paragraph styles** for consistency
- Apply styles to all similar elements at once

### Problem: Can't add signature

**Solution**:
1. Print the document
2. Sign by hand
3. Scan the signed page
4. Insert scanned signature as image in Google Docs:
   - **Insert → Image → Upload from computer**
   - Position signature in verification section

**Alternative**: Use Google Docs add-ons for digital signatures:
- DocuSign for Google Docs
- HelloSign (now Dropbox Sign)

### Problem: File size too large for upload

**Solution**:
- Download as PDF (usually smaller than DOCX)
- Compress PDF using online tools (SmallPDF, Adobe Acrobat)
- Remove high-resolution images if any

## Best Practices

1. **Keep it updated**: Update GPA after each semester
2. **Consistent formatting**: Use same fonts, spacing throughout
3. **Proofread carefully**: Check for typos, calculation errors
4. **Save multiple versions**: Keep draft, final, and submitted versions
5. **Print preview**: Always check **File → Print** to see how it will look printed
6. **Professional appearance**: Clean, organized, easy to read
