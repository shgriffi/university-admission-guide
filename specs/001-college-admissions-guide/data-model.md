# Content Outline & Information Architecture

**Feature**: 001-college-admissions-guide
**Purpose**: Define the structure, organization, and templates for all guide content

---

## Document Structure

### Overall Organization

The guide follows a **chronological then topical** structure:
1. Introduction and overview (how to use the guide)
2. Timeline-based content (what to do when)
3. Topic-based deep dives (universities, careers, homeschool requirements, visits)
4. Reference materials (appendices, sources, glossary)

**Total estimated length**: 80-120 pages

## Content Guidelines

### Writing Standards

**Reading Level**:
- Target: 10th grade Flesch-Kincaid or below
- Test all sections with readability tools before finalizing
- Simplify complex sentences
- Define technical terms on first use

**Voice and Tone**:
- Second person ("you" and "your family") for direct engagement
- Active voice preferred over passive
- Clear, straightforward, jargon-free
- Encouraging but realistic (don't overpromise)
- Non-judgmental (family's choices are valid)

**Sentence and Paragraph Structure**:
- Average sentence length: 15-20 words
- Vary sentence length for readability
- Paragraphs: 3-5 sentences typically
- Use bullets and numbered lists for multi-step processes
- Break up long sections with subheadings

**Examples and Concreteness**:
- Every abstract concept must include a concrete example
- Use Connor's specific situation in examples when relevant
- Real deadlines, real university names, real career scenarios
- Avoid vague language ("soon," "a lot," "many") - use specific numbers and dates

### Formatting Standards

**Headings**:
- H1: Section titles (e.g., "Section 3: University Profiles")
- H2: Major subsections (e.g., "Duke University")
- H3: Components within subsections (e.g., "Application Requirements")
- H4: Details (e.g., "Homeschool-Specific Requirements")
- Never skip heading levels

**Lists**:
- Bulleted lists for unordered items
- Numbered lists for sequential steps or priorities
- Checkboxes for actionable items

**Tables**:
- Use for comparisons (university deadlines, career salaries, program availability)
- Keep tables simple - max 5-6 columns for readability
- Always include a header row
- Alt text for accessibility if PDF

**Cross-References**:
- Format: "See Section 3.2 for Duke's homeschool requirements"
- Hyperlink in digital versions
- Page number references in printed versions
- Bi-directional links where relevant

**Emphasis**:
- **Bold** for important terms, deadlines, warnings
- *Italic* for emphasis or examples
- ALL CAPS only for critical warnings (rarely)
- Underline for links only (no other use)

### Citation Format

**In-Text Citations**:
```
According to the Bureau of Labor Statistics (verified 01/15/2026), aerospace engineers earn a median salary of $122,270 per year.
```

**Source Verification**:
Every fact must include:
- Source name
- Verification date
- Link (in digital version) or URL (in printed version)

**Source List Format** (in Section 8.2):
```
- [Source Name]: [Brief description]
  URL: [Full URL]
  Last Accessed: [MM/DD/YYYY]
  Use: [What information was sourced from here]
```
